# Tic-Tac-Toe Project (Fullstack)

Добро пожаловать! Это демонстрационный учебный проект игры **"Крестики-нолики"**, реализованный как полноценное Fullstack-приложение.

## 🚀 Демонстрация

### ⏳ Важное примечание (Холодный старт)
Проект развернут на бесплатном тарифе **Render**. 
> **Пожалуйста, наберитесь терпения:** если приложением никто не пользовался последние 15 минут, сервер «засыпает». Первый запрос (авторизация или загрузка данных) может занять **30–50 секунд**, пока бэкенд просыпается. После этого всё будет работать без задержек! ☕ 

### 🕹️ Быстрый старт (Тестовые аккаунты)

Чтобы не тратить время на регистрацию, используйте данные готовых пользователей. 
Для игры со вторым игроком откройте вторую вкладку в **режиме инкогнито** или в другом браузере.

| Роль | Логин | Пароль |
| :--- | :--- | :--- |
| **Игрок 1** | `first-player` | `first-player` |
| **Игрок 2** | `second-player` | `second-player` |

> Если при входе возникла задержка, значит сервер просыпается (см. примечание о «Холодном старте»).

**[Играть в браузере](https://lindsyro.github.io/ttt-vue)**

### Стек технологий

#### Backend
*   **Nest.js** — современный Node.js фреймворк
*   **Sequelize ORM** — работа с базой данных
*   Деплой: **Render** 

#### Database
*   **PostgreSQL** — развернута на облачной платформе **Supabase**
*   Автоматическая синхронизация таблиц при деплое

#### Frontend
*   **Vue 3** (Composition API) + **Vite**
*   **Vuex** для управления состоянием
*   **Axios** для связи с API
*   Деплой: **GitHub Pages** (через GitHub Actions)
---

### 📂 Структура проекта
*   `/api` — Серверная логика, API эндпоинты и работа с БД.
*   `/frontend` — Клиентское приложение на Vue.
*   `/.github/workflows` — Сценарии автоматического деплоя.
